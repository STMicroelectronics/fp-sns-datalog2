Purpose

The BLE Manager Library is an STM32 middleware providing an APIs to manage the BLE service according to the BlueNRG-MS, BlueNRG-1/2, BlueNRG-LP and WB middleware APIs. Parson (lighweight json library written in C) must been included to parse the extended configuration characteristic commands.
