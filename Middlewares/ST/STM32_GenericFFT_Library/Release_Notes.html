Purpose

The GenericFFT Library is an STM32 middleware providing a set of APIs to calculate the Fast Fourier Transform. It is based on CMSIS implementation.

Main features:

  • Possibility to allocate memory through malloc/free
  • Memory allocation from user space or automatically setup by the library
  • Framing and windowing
  • Direct processing
