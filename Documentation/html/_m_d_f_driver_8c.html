◆ MDFDrvSetDataBuffer()

sys_error_code_t MDFDrvSetDataBuffer ( MDFDriver_t *  _this,
int16_t *  p_buffer,
uint32_t  buffer_size 
)

Set the buffer that will be filled by the driver with audio data from the sensor. The application must provide a data buffer before starting the driver.

Parameters
_this[IN] specifies an instance of the driver.
p_buffer[IN] specifies a pointer to the data buffer.
buffer_size[IN] specifies the size in word of the data buffer.
Returns
SYS_NO_ERROR_CODE if success, an error code otherwise.
