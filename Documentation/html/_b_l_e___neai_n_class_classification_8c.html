This function is called when there is a change on the gatt attribute With this function it's possible to understand if Piano is subscribed or not to the one service.

Parameters
void*VoidCharPointer
uint16_tattr_handle Handle of the attribute
uint16_tOffset: (SoC mode) the offset is never used and it is always 0. Network coprocessor mode:
  • Bits 0-14: offset of the reported value inside the attribute.
  • Bit 15: if the entire value of the attribute does not fit inside a single ACI_GATT_ATTRIBUTE_MODIFIED_EVENT event, this bit is set to 1 to notify that other ACI_GATT_ATTRIBUTE_MODIFIED_EVENT events will follow to report the remaining value.
uint8_tdata_length length of the data
uint8_t*att_data attribute data

tBleStatus BLE_Neai1ClassClassificationUpdate ( BLE_1CC_output_t  output)

Update NEAI Classification Characteristic Value.

Update classification characteristic value with 1-Class library output.

Parameters
BLE_1CC_output_toutput contains info about:
  • phase = (idle=0x00) | (classification=0x01)
  • state = (NEAI_OK=0x00) | .....
  • is_outlier = 0x01 if it is an outlier, 0x00 otherwise

ONLY THE PHASE FIELD IS MANDATORY

Return values
tBleStatusStatus

◆ BLE_NeaiNClassClassificationUpdate()

tBleStatus BLE_NeaiNClassClassificationUpdate ( BLE_NCC_output_t  output)

Update NEAI Classification Characteristic Value.

Update classification characteristic value with N-Class library output.

Parameters
BLE_NCC_output_toutput contains info about:
  • phase = (idle=0x00) | (classification=0x01)
  • state = (NEAI_OK=0x00) | .....
  • major class = from 0x00 to CLASS_NUMBER_NCC
  • probabilities = array with size equals to CLASS_NUMBER_NCC that contains class probabilities

ONLY THE PHASE FIELD IS MANDATORY

Return values
tBleStatusStatus
