Init environmental info service.

Parameters

uint8_tHumEnable: 1 for enabling the BLE humidity feature, 0 otherwise.
uint8_tNumTempEnabled: 0 for disabling BLE temperature feature 1 for enabling only one BLE temperature feature 2 for enabling two BLE temperatures feature
Return values
