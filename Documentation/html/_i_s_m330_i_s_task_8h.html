AManagedTaskEx* ISM330ISTaskAlloc ( const void *  pIRQConfig,
const void *  pISPUConfig,
const void *  pCSConfig 
)

Allocate an instance of ISM330ISTask in the system heap.

Parameters
pIRQConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO connected to the ISM330IS sensor and configured in EXTI mode. If it is NULL then the sensor is configured in polling mode.
pISPUConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO connected to the ISM330IS ISPU and configured in EXTI mode.
pCSConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO identifying the SPI CS Pin.
Returns
a pointer to the generic object AManagedTaskEx if success, or NULL if out of memory error occurs.
