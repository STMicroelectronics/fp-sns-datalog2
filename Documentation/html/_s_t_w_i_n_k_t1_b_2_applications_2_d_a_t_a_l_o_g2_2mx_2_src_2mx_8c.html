Detailed Description

Project peripherals configuration files.

Author
SRA - GPM
Date
Apr 9, 2021

This files defines the peripherals configuration parameters that can be used in the Low Level API of an eLooM driver - IDrvInit(). It creates a link between the initialization code generated by CubeMX (see the .ioc file in the tools folder) and the eLooM framework.
