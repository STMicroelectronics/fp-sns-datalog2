Detailed Description

An event that encapsulate a data of type EMData_t.

Author
STMicroelectronics - AIS - MCD Team
Version
M.m.b
Date
May 13, 2022

This kind of events are generated by data source implementing the ::IDataEvent_tSrc interface. To receive this evens an object has to implement the ::IDataEvent_tListener interface and register itself with a source of data event.
