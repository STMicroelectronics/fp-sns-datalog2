This event shall only be generated if scanning was enabled using the LE Set Extended Scan Enable command. It reports advertising events using either legacy or extended advertising PDUs. The Controller may split the data from a single advertisement (whether one PDU or several) into several reports. When a scan response is received, bits 0-2 and 4 of the event type shall indicate the properties of the original advertising event. An Event_Type with a data status field of "incomplete, data truncated" indicates that the Controller attempted to receive an AUX_CHAIN_IND PDU but was not successful.

WEAK_FUNCTION ( void   hci_disconnection_complete_eventuint8_t Status, uint16_t Connection_Handle, uint8_t Reason) 
@brief The Disconnection Complete event occurs when a connection is terminated.
