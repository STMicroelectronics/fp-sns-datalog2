Update NeaiAnomalyDetection characteristic value.

Parameters

uint8_tstate: library result of current phase
uint8_tcompletion_percentage:
BLE_AD_output_toutput:

Return values
tBleStatusStatus

Update NeaiAnomalyDetection characteristic value.

Parameters
BLE_AD_output_toutput contains info about:
  • phase = (idle=0x00) | (learning=0x01) | (detecting=0x02)
  • state = (NEAI_OK=0x00) | .....
  • progress = from 0x00 to 0x64 (completion percentage)
  • status = (normal=0x00) | (anomaly=0x01)
  • similarity = from 0x00 to 0x64 (inference probability)

ONLY THE PHASE FIELD IS MANDATORY if you don't want use one of the others info, you can put NEAI_AD_ESCAPE

tBleStatusStatus
