◆ DataBuffAllocator_f

typedef uint8_t *(* DataBuffAllocator_f) (IDataBuilder_t *_this, void *p_data_build_context)

Allocator used by the data builder to allocate a new buffer in the build context. It is used during the new input data processing when all these conditions are true:

  • The target data is ready.
  • There are element in the input data not processed.
  • The build strategy is E_IDB_NO_DATA_LOSS.
Parameters
_this[IN] specifies a pointer to the object.
p_data_build_context[IN] specifies a context for the data build operation. This parameter is not used by the data builders IF, but is for exclusive use of the caller.
Returns
a new data buffer if success, NULL otherwise.

◆ IDataBuilder_Reset()

static sys_error_code_t IDataBuilder_Reset ( IDataBuilder_t *  _this,
void *  p_data_build_context 
)
inlinestatic

Reset the interface. Called at the beginning of a new data creation. The object reset its internal state to get ready to receive new input data used to build the target data.

Parameters
_this[IN] specifies a pointer to the object.
p_data_build_context[IN] specifies a build context. It is provide by the object that uses the data builder interface. The interface must use this context when it calls the data ready callback.
Returns
SYS_NO_ERROR_CODE if success, an error code otherwise.
