Detailed Description

Global System configuration file.

Author
SRA

This file include some configuration parameters grouped here for user convenience. This file override the default configuration value, and it is used in the "Preinclude file" section of the "compiler > prepocessor" options. This file must be moved in the application folder by the user and renamed 'SensorManager_conf'
