Allocate an instance of ILPS22QSTask in the system heap.

Parameters
pIRQConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO connected to the ILPS22QS sensor and configured in EXTI mode. If it is NULL then the sensor is configured in polling mode.
