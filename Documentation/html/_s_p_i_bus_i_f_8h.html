Initialize a sensor object. It must be called once before using the sensor.

Parameters
_this[IN] specifies a sensor object.
nWhoAmI[IN] specifies the sensor ID. It can be zero.
pxSSPinPort[IN] specifies the port PIN of the Slave Select line.
nSSPin[IN] specifies the pin number of the Slave Select line.
nAutoInc[IN] specifies the SPI address auto-increment to allow multiple data read/write (see sensor datasheet: if needed set 0x40, else 0)
Returns
SYS_NO_EROR_CODE if success, an error code otherwise.
