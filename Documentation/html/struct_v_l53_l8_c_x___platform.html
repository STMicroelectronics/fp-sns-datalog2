Detailed Description

Structure VL53L8CX_Platform needs to be filled by the customer, depending on his platform. At least, it contains the VL53L8CX I2C address. Some additional fields can be added, as descriptors, or platform dependencies. Anything added into this structure is visible into the platform layer.
