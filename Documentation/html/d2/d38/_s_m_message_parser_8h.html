Detailed Description

Input Output and Feature report Declaration.

Author
SRA - MCD

This file declare the Input, Output, Feature and Internal report for the application. The reports are generated by the sensors as well as form the MCU (internal reports).

Report ID list:

  • 0x01: inputReport (Sensor --> MCU). ISM330DHCX
  • 0xFE: internalReport (MCU --> MCU). FORCE_STEP
