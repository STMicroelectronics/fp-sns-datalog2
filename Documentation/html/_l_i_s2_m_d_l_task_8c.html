Allocate an instance of LIS2MDLTask in the system heap.

Parameters
pIRQConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO connected to the LIS2MDL sensor and configured in EXTI mode. If it is NULL then the sensor is configured in polling mode.

Returns
a pointer to the generic object AManagedTaskEx if success, or NULL if out of memory error occurs.
