Detailed Description

Declare the Power Mode State Machine events, state, and status.

Author
SRA This file declares:
  • the events that define the transactions in the Power Mode State Machine.
  • the states of the Power Mode State Machine
  • the status of the Power Mode state Machine. It is a bit of information linked to the PM State Machine available system wide using the API SysGetPowerStatus().

All these three element together define the Power Mode State Machine. This definition is application dependent and it is separate by the API to operate the PM State Machine. The API are defined by the Application Power Mode Helper object instance (IAppPowerModeHelper > AppPowerModeHelper).
