Detailed Description

Application specific error code.

Author
SRA

Application defines its own error code in this file starting form the constant APP_BASE_ERROR_CODE. It is recommended to group the error code in the following groups:

  • Low Level API error code
  • Service Level error code
  • Task Level error code
