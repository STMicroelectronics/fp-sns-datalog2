This function is called when there is a change on the gatt attribute With this function it's possible to understand if Machine Learning Core is subscribed or not to the one service.

Parameters
void*VoidCharPointer
uint16_tattr_handle Handle of the attribute
uint16_tOffset: (SoC mode) the offset is never used and it is always 0. Network coprocessor mode:
  • Bits 0-14: offset of the reported value inside the attribute.
  • Bit 15: if the entire value of the attribute does not fit inside a single ACI_GATT_ATTRIBUTE_MODIFIED_EVENT event, this bit is set to 1 to notify that other ACI_GATT_ATTRIBUTE_MODIFIED_EVENT events will follow to report the remaining value.
uint8_tdata_length length of the data
uint8_t*att_data attribute data
Return values
None
