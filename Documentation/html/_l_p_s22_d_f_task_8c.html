Initialize the hardware resource for the task. This task doesn't need a driver extending the IDriver interface because:

  • it manages two GPIO pins, that are the CS connected to the sensor SPI IF and the EXTI line.
  • it uses the common sensor driver provided by the ST Sensor Solutions Software Team .
Parameters
_this[IN] specifies a task object.
pParams[IN] specifies task specific parameters. Not used
Returns
SYS_NO_ERROR_CODE if success, an error code otherwise.

◆ LPS22DFTaskAlloc()

AManagedTaskEx* LPS22DFTaskAlloc ( const void *  pIRQConfig,
const void *  pCSConfig 
)

Allocate an instance of LPS22DFTask in the system heap.

Parameters
pIRQConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO connected to the LPS22DF sensor and configured in EXTI mode. If it is NULL then the sensor is configured in polling mode.
pCSConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO identifying the SPI CS Pin.
Returns
a pointer to the generic object AManagedTaskEx if success, or NULL if out of memory error occurs.
