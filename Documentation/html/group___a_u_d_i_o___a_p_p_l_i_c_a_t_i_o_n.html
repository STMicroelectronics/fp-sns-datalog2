static void TukeyWin ( uint32_t  len,
float32_t  ratio,
float32_t *  dest 
)
static

@Brief: Tukey window function

@Description: 

   /
   |  1/2 * { 1 + cos( 2PI/r * [ x - r/2 ]) },     0 ≤ 1 < r/2
     |

f(x) = < 1, r/2 ≤ x < 1 - r/2 | | 1/2 * { 1 + cos( 2PI/r * [x - 1 + r/2]) }, 1 - r/2 ≤ x < 1 \

with: r = ratio x = normalized input. In the implementation below x is equal to @tukeyWin_in

ref: https://www.mathworks.com/help/signal/ref/tukeywin.html

Parameters
[in]lendestination array dimension
[in]ratioCosine fraction, specified as a real scalar. The Tukey window is a rectangular window with the first and last r/2 percent of the samples equal to parts of a cosine.
[out]destpointer to output array containing the Tukey window
