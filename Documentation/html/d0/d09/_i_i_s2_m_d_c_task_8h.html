AManagedTaskEx * IIS2MDCTaskAlloc ( const void *  pIRQConfig,

)

Allocate an instance of IIS2MDCTask in the system heap.

Parameters
pIRQConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO connected to the IIS2MDC sensor and configured in EXTI mode. If it is NULL then the sensor is configured in polling mode.
pCSConfig[IN] specifies a MX_GPIOParams_t instance declared in the mx.h file. It must be a GPIO identifying the SPI CS Pin.
Returns
a pointer to the generic object AManagedTaskEx if success, or NULL if out of memory error occurs.
