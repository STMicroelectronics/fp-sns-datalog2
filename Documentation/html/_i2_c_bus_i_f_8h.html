ABusIF * I2CBusIFAlloc ( uint16_t  who_am_i,
uint8_t  address,
uint8_t  auto_inc 
)

Initialize a sensor object. It must be called once before using the sensor.

Parameters
_this[IN] specifies a sensor object.
nWhoAmI[IN] specifies the sensor ID. It can be zero.
nAutoInc[IN] specifies the I2C address auto-increment to allow multiple data read/write (see sensor datasheet: if needed set 0x80, else 0)
Returns
SYS_NO_EROR_CODE if success, an error code otherwise.
